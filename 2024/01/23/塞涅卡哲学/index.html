塞涅卡哲学

观默2024-01-232024-02-01

心得

关于财富

  1. 财富并不能让人幸福，更多情况下，获得财富后只是换了一种烦恼。
  2. 应该尝试体验贫穷，让自己用极少的花费去生活。
  3. 财富会让我们傲慢自大且不知羞耻。
  4. 钱财足以为生的人就是真正的富人，而致富的最快途径就是放弃对财富无休止的追求。

降低欲望

  1. 从实用性角度看待我们的需求，应该提防没有经过深思熟虑的被欲望引诱。
  2. 购买那些对于你来说必要的东西，而不是那些你需要的东西。那些不必要的东西，即使一小部分也会带来巨大的负担。

思考死亡

  1. 让我们不断地思考一下自己的死亡，就像那些我们所爱的人的死亡一样。
  2. 生活就像一场演出，重要的不在于上台演了多久，而在于演得有多好。演出不管在什么时候停止，都是一样的。你可以随时选择终止，只要确保终止前演得精彩就可以了。
  3. 我们不应该追求活得更长久，而应该努力活得更正确、充实。因为想活得长久只需要命运，而要活得正确、充实，则需要灵魂。
  4. 把每一天都当作生命的最后一天。

关于读书

  1. 无目标的多读书没有用，赢得博学的称赞又有什么益处呢，却为此付出了大量时间。因此要读为数不多的好书。

生活与工作

  1. 我们不应该推迟现在的生活，希望有一天我们能够退休后过上一直梦想的生活。
  2. 找到目标，一路前行。而不是一路观光，四处游行。
  3. 你还必须思考，自己的性格是更适合实际的活动还是安静的研究和沉思，然后选择符合自己天赋和天性的方向。
  4. 凡事都应该追随本心，并且表里如一

快乐与自由

  1. 快乐是你的目标，但你偏离了方向！你认为你会带着财富和荣誉到达那里。这意味着你在焦虑中寻找快乐！你追逐这些东西，好像它们会带来快乐，而事实上，它们是痛苦的源泉。

处理负面情绪

  1. 审视引起我们如此不安的内心判断并对内在判断以及这些判断导致的情绪进行观测。
  2. 愤怒是短暂的精神错乱。
  3. 治疗愤怒的最好方法是拖延。从愤怒一开始就要求它放慢进程。这样做不是因为它会原谅，而是因为它会进行评估。虽然愤怒最初的攻势很猛，但如果被迫等待，它将撤退。但不要试图一下子毁掉它。如果一点一点地消磨，它就会被击败。

面对苦难

  1. 重要的不是我们遭受了什么，而是我们如何应对它(心态)。
  2. 遇到苦难时的良好品质帮助我们成长。
  3. 所以我们并不歌颂苦难，而要歌颂在苦难中不屈的灵魂。
  4. 苦难让我们成长，顺境却会让人沉沦，苦难塑造品质。

书摘

《论生命之短暂》

  • 因此，不敢回忆过去的人，往往都是贪得无厌、心高气傲、小人得志、背信弃义、巧取豪夺、挥霍无度之辈。可是，过去才是生命中最神圣、最独立的光阴。唯有过去能超越人类一切风险，不受命运的摆布，不被欲望、恐惧或疾病扰乱。过去是屹立不倒、永恒存在的财产，任何事都不能将其打扰或夺走。

  • 无论富裕还是贫苦，人总会有不安的理由。生活就这样，被一个又一个渴望推着向前走。我们永远在渴望悠闲，却又从未真正享受过悠闲。

  • 我们必须明白，困境来源于自身，而不是周围的环境。因为自身软弱，我们才没有办法长久忍受任何事，无论是劳苦、欢乐还是我们自己。

  • 你还必须思考，自己的性格是更适合实际的活动还是安静的研究和沉思，然后选择符合自己天赋和天性的方向。

  • 我们还应学会变通，不将希望过多寄托在原计划上，而是根据实际情况灵活调整目标，避免在目标或现实状况发生改变时心生恐慌。

  • 正义之士啊，你们何必抱怨？我用虚假的幸福包裹别人，用缥缈漫长的梦欺骗他们无知的心灵，用金银象牙装扮他们的外表，但他们的内心却空空如也。

  • 幸福是与内在本性和谐一致的生活方式

  • 关注能改善生活的东西，但又不过度沉迷其中

《与塞涅卡共进早餐》

  • 最深厚的友谊是建立在相互钦佩基础上的，每个人都能在对方的品质中看到他或她所钦佩的东西。

  • 我们不应该推迟现在的生活，希望有一天我们能够退休后过上一直梦想的生活。

  • 审视引起我们如此不安的内心判断

  • 减少担忧的第一个也最有效的方法之一，就是在你产生内在判断，和你对未来的事情感到担忧的时候，对你的内在判断以及这些判断导致的情绪进行观测

  • 愤怒是一种短暂的精神错乱

  • 愤怒本身总是基于两种心理判断。第一个判断是“我受到了伤害”或“有人对我不公”。第二个判断是“如果我受到了伤害，我应该通过报复或惩罚来寻求补偿”。如果这两个判断结合起来，结果很可能表现为极端愤怒。

  • 治疗愤怒的最好方法是拖延。从愤怒一开始就要求它放慢进程。这样做不是因为它会原谅，而是因为它会进行评估。虽然愤怒最初的攻势很猛，但如果被迫等待，它将撤退。但不要试图一下子毁掉它。如果一点一点地消磨，它就会被击败。

  • 我们需要停下来，仔细对印象进行审视，然后决定或“选择”它们是否应该被接受。

  • 智者面对别人犯错误时，在精神上是平静和平衡的。他不是犯错者的敌人，而是帮助犯错者康复。每天，他离开家时都会想：“今天，我会遇到许多嗜酒成性的人，许多被欲望所征服的人，许多缺乏感恩之心的人，许多被贪婪所奴役的人，还有许多被野心的虚假承诺所迷惑的人。”但所有这些情况，他都会仁慈地对待，正如医生对待自己的病人一样。

  • 重要的不是我们遭受了什么，而是我们如何应对遭遇。任何逆境都可能袭击我们，但如何应对才是衡量我们真实品质的标准。

  • 所谓对逆境进行预演，其基本思想是，在你的想象中对任何可能发生的不幸事件进行简单的演练

  • 不要希望事情按照你的意愿发生，而是希望事情顺其自然地发生。这样你的生活就会平静。

  • 对许多人来说，获得财富并不是他们麻烦的结束，而只是他们麻烦形式上的改变

  • 我们应该记住，没有财富的痛苦比失去财富的痛苦要小得多。然后，我们会意识到，既然贫穷可失去的甚少，那么折磨我们的可能性就小得多

  • 对塞涅卡来说，钱财足以为生的人就是真正的富人，而致富的最快途径就是放弃对财富无休止的追求。

  • 简单的生活方式可以减少开支，减少你需要工作的时间，从而获得更多的自由时间。

  • 在斯多葛学派看来，我们必须了解自己的特点，这样才不会与本性抗争，试图追求超出我们能力范围的东西。如果你不了解自己是谁，就不可能拥有自我意识，过真实的生活。最后，如果你试图模仿别人而忽视自己的本性，那么你就不可能过上真实而幸福的生活。

  • 首先，从对死亡的恐惧中解脱出来……然后把自己从对贫穷的恐惧中解放出来。

  • 让一个人生命圆满的是生命的质量，而不是长度。

  • 把每一天都当作生命的最后一天

  • 让我们不断地想到我们自己的死亡和我们所爱的人的死亡……任何时候可能发生的事情，都可能在今天发生。”

  • 自由意味着不被以下事物所奴役：错误的判断、极端的负面情绪、愤怒、强烈的冲动、不快乐、对未来的焦虑、对外部事物的渴望、情感受伤的感觉以及其他人的意见或行为。

  • 快乐是你的目标，但你偏离了方向！你认为你会带着财富和荣誉到达那里。这意味着你在焦虑中寻找快乐！你追逐这些东西，好像它们会带来快乐，而事实上，它们是痛苦的源泉。

《塞涅卡道德书简》

  • 你需要阅读的是有限数量的大师级思想家的作品，且消化它们。

  • 知足的贫穷是一份尊荣的资产。

  • 没有理由让自我炫耀的欲望，引诱你去广泛地宣传，导致你去公众面前讲座或高谈阔论。

  • 比起现实，我们在想象中会遭受更多苦难。

  • 我们选择不去贬低他人时，也应避免让他人认为我们是可以被贬低的。

  • 财富，已经阻止很多人获取智慧；贫穷，则是无拘无束的。

  • 如果你想在精神上得到空闲的话，要么去做个穷人，要么就去模仿穷人吧。

  • 除非你努力地去实现朴素的生活方式，否则不管如何学习也无济于事。而朴素的生活就是自甘贫穷。放弃所有的那些借口，

  • 对于很多人来说，获得财富都不是一个终点，而是一次与烦恼的交换。

  • 出几天时间，从日常事务中抽身，并让自己用最吝啬的费用去生活。与贫穷建立往来关系吧。

  • 人们不关心他们活得多么高尚，而只是关心活了多久。然而，能活多久并不是人们可以掌控的，高尚地去生活却是每个人都力所能及的。

  • 工作不是一个好的东西。那么，什么才是好的？我认为对工作的蔑视才是。

  • 没有人知道自己是何时开始渴望那些他所渴望的东西的。因为没有经过深思熟虑，他就被欲望所引诱

  • 实用性可以用来衡量我们的需求。但是，采用什么标准，才能测试其是否已经过度了呢？这就是为什么人们会陷入享乐中，当他们养成习惯后，就会难以自拔。这也是为什么人们会如此可怜，因为他们已经超过标线了——那些本是多余的东西，却成了他们必不可缺的东西。

  • 与其说享受快乐，不如说他们已成了享乐的奴隶。

  • 意一下那些失去时令我们痛哭流涕和为之疯狂的东西。你会发现，令我们烦恼的并不是那些失去的东西本身，而是失去这个概念。

  • 有限的阅读可以带来好处，不加限制地去读各种各样的书，只是满足乐趣。

  • 你难道不会努力让所有人都明白，去追寻多余的东西意味着消耗大量的时间；并且，很多人都仅仅是在积累生活手段的过程中，度过一生的吗？想想个别的人，且观察一下普通人，没有谁不渴望明天。(13)“其中会有什么伤害呢？”你会问。那是无尽的伤害，因为这些人不是在生活，而一直在准备生活。

  • 像你希望优于你的人对待你那样，去对待那些不如你的人。

  • 节俭的生活可以让人活得很久

  • 真正的喜悦永远不会停止，且永远不会转换为悲伤。

  • 我们最大的阻碍，就是过于满足于自我了。如果有人称我们是好人，或是明智和高尚的，我们就会自以为是。我们并不会因为适度的赞美而满意，而是会接受一切无耻的奉承，就像理所当然一样。

  • 现在你一直关注自己的苦楚，一旦你停止关注自我，所遭受的悲痛就会消散。

  • 因为哀悼根本不被认为是一种高尚品德的表现。

  • 让我们不断地思考一下自己的死亡，就像那些我们所爱的人的死亡一样。

  • 也许刚刚离我们而去的人，只是被提前送到目的地，正在等着我们的到来。

  • 不要张扬你的退休

  • 必要给自己贴个“哲学家和寂静主义者”的标签。可以用其他名称来标榜你退休的目的，只需称健康和身体虚弱，或仅仅是懒惰。去吹捧退休，只是毫无价值的自我追求罢了。

  • 如果一个人无穷无尽地去渴望所有东西的话，他又怎会去思考自己的死亡呢？

  • 贪婪、野心，以及对死亡的恐惧——是它们把那些征服世界的征服者都打败了。

  • 最为明智的人一看到那些施舍的东西 ￼，就会马上离开。因为他知道，即使为了很小的恩惠，人们也会付出很大的代价。

  • 生活就像一场演出，重要的不在于上台演了多久，而在于演得有多好。演出不管在什么时候停止，都是一样的。你可以随时选择终止，只要确保终止前演得精彩就可以了。

  • 我们经常听到如下的赞美其生活的方式：“谁谁的生活很奢侈。”但他们的意思是：“那个人被奢侈弄得很软弱。”

  • 没有人会回顾或反省自己的生活。我们的想法全都专注于我们即将要做的，而不是已经做的；但是，我们未来的计划却经常取决于过去。

  • 我们不应该把自己限制于写作或阅读中，持续写作会令我们筋疲力尽，而持续阅读会令我们软弱和松懈。如果我们将两者交汇融合，阅读所丧失掉的成果，可能会被手中的笔所填充。

  • 而财富会让人不知羞耻。美好的东西会令我们的灵魂伟大，而财富会令我们傲慢。傲慢只不过是虚伪地去展现伟大。

  • 任何只为了赚钱的学习都不是好的学习，我不会崇敬这些学习，也不认为它们是好的。

  • 如果一位不良的邻居仅仅侵占了我的一小块儿土地，我就一直怀恨在心，那么精确衡量每一寸土地并探测每一丝误差，又有什么用呢？数学家教会我如何不失边界，然而，我想知道的是如何轻松愉快地放弃所有边界。

  • 赢得像“你是多么博学的人啊！”这样的称赞，会耗费我们巨大的时间，且会让其他人听不惯。

  • 奢侈忽略了自然的诉求，每一天都在人群中拓展其力量，散播着恶习。首先，奢侈会将对于大自然来说多余的东西作为渴求的对象，然后追求那违背自然的东西。最终，其会使灵魂成为身躯的奴隶，被躯体的欲望彻底征服。

  • 我们不应该追求活得更长久，而应该努力活得更正确、充实。因为想活得长久只需要命运，而要活得正确、充实，则需要灵魂。如果活得充实的话，一个人的一生真的很长；

  • 购买那些对于你来说必要的东西，而不是那些你需要的东西。那些不必要的东西，即使一小部分也会带来巨大的负担。

  • 我们要拒绝享乐，因为它会令我们虚弱且缺乏阳刚之气。它会极大地要求我们，而且，会令我们去祈求命运。其次，我们必须蔑视财富。财富是奴役的证书。放弃金银，及家里其他一切的豪华装饰。只有付出，才能获得自由。如果你将自由视作珍贵的东西，就必须低估其他的一切。再见。

  • 想拥有越多越好，还是拥有足够多就可以了？那些想要更多的人，渴望也就更多——这就证明了他获取的还不够多。而那些知道满足的人，其能够获取富人永远不会获取的东西——一个停止点
